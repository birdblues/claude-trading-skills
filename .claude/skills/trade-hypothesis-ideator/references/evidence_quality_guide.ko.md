# Evidence Quality Guide

증거 근거 항목을 구성할 때 이 간단한 rubric을 사용하세요.

## Source Weights (정성)
- High: 검증된 trade log, 재현 가능한 backtest 결과, 감사 가능한 fill 데이터
- Medium: 결과가 명시된 구조화된 journal 스니펫, 반복 관찰된 차트 패턴
- Low: 일회성 anecdote, 제약 없는 직관, 재현 불가능한 스크린샷

## 체크리스트
- 구체적인 source로 추적 가능한 증거를 우선하세요.
- signal과 레짐 특이적 noise를 구분하세요.
- kill criteria에 반증 가능한 counter-condition을 최소 1개 포함하세요.
- 사용 불가능한 feature나 지원되지 않는 데이터 granularity가 필요한 가설에는 감점을 적용하세요.
