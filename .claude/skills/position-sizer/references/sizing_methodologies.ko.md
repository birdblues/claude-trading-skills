# 포지션 사이징 방법론

## 개요

포지션 사이징은 각 트레이드에서 몇 주를 매수할지 결정합니다. 올바른 사이징은 장기적인 포트폴리오 생존에서 가장 중요한 요소입니다. 훌륭한 종목 선택도 잘못된 사이징이면 계좌를 망칠 수 있고, 평범한 선택도 적절한 사이징이면 다음 기회를 위한 자본을 지킬 수 있습니다.

이 참고 문서는 세 가지 핵심 방법인 Fixed Fractional, ATR-based, Kelly Criterion을 다룹니다. 각 방법은 고유한 강점과 이상적인 사용 사례를 가집니다.

---

## Fixed Fractional Method (Percentage Risk)

### 개념

모든 트레이드에서 계좌의 고정 비율을 리스크로 설정합니다. Van Tharp가 대중화했고 Mark Minervini와 William O'Neil이 엄격하게 적용한, 프로 트레이더 사이에서 가장 널리 쓰이는 방법입니다.

### 공식

```
risk_per_share = entry_price - stop_price
dollar_risk    = account_size * risk_pct / 100
shares         = int(dollar_risk / risk_per_share)
```

### 표준 리스크 수준

| Risk % | 트레이더 프로필 | 비고 |
|--------|---------------|-------|
| 0.25-0.50% | 보수적 / 대형 계좌 | 기관급 리스크 |
| 0.50-1.00% | 숙련된 스윙 트레이더 | Minervini 권장 범위 |
| 1.00-1.50% | 활발한 트레이더, 검증된 edge | 테스트된 시스템의 표준 |
| 1.50-2.00% | 공격적, 높은 승률 | 대부분 전략의 최대치 |
| > 2.00% | 위험함 | 파산 리스크가 빠르게 증가 |

### 예시

- 계좌: $100,000
- 진입가: $155.00, 손절가: $148.50
- 주당 리스크: $6.50
- 1% 리스크일 때: $1,000 / $6.50 = **153 shares**
- 포지션 가치: $23,715 (계좌의 23.7%)

### 사용 시점

- 대부분의 스윙/포지션 트레이드 기본 방법
- 명확한 기술적 손절 레벨(지지선, 이동평균, 직전 저점)이 있을 때
- 리스크 파라미터가 확립된 시스템을 거래할 때

### Minervini / O'Neil 통합

Mark Minervini 권장사항:
- 랠리 초기 단계에서는 트레이드당 1% 초과 리스크 금지
- 연속 손실 후 0.5%로 축소
- "progressive exposure" 모델 사용: 반 포지션으로 시작 후 확인 시 추가
- 최대 portfolio heat(총 오픈 리스크): 6-8%

William O'Neil 권장사항:
- 매수가 대비 7-8% 하락 시 손절(절대 상한)
- 숙련 트레이더의 선호 손절 폭: 3-5%
- 노출 확대 전 시장 방향 확인을 위해 "follow-through day" 사용

## ATR-Based Method (Volatility Sizing)

### 개념

Average True Range(ATR)로 손절 거리를 설정해, 종목의 변동성에 맞게 포지션 크기를 자동 조정합니다. Turtle Traders(Richard Dennis, 1983)에서 시작된 방식입니다.

### 공식

```
stop_distance  = atr * atr_multiplier
stop_price     = entry_price - stop_distance
risk_per_share = stop_distance
dollar_risk    = account_size * risk_pct / 100
shares         = int(dollar_risk / risk_per_share)
```

### ATR 배수 가이드

| Multiplier | 손절 폭 | 스타일 |
|-----------|-----------|-------|
| 1.0x | 좁음 | 데이트레이딩, 초단기 |
| 1.5x | 중간-좁음 | 스윙 트레이딩, 2-5일 보유 |
| 2.0x | 표준 | 대부분의 스윙 트레이드 기본값 (Turtle Traders) |
| 2.5x | 넓음 | 포지션 트레이딩, 2-8주 보유 |
| 3.0x | 매우 넓음 | 추세추종, 수개월 보유 |

### 예시

- 계좌: $100,000, 리스크: 1%
- 진입가: $155.00, ATR(14) = $3.20, Multiplier = 2.0x
- 손절 거리: $6.40, 손절가: $148.60
- 금액 리스크: $1,000
- 주식 수: int($1,000 / $6.40) = **156 shares**

### 사용 시점

- 서로 다른 종목 간 변동성 조정 사이징이 필요할 때
- 명확한 지지/저항이 없어 이산적인 손절가를 정하기 어려울 때
- 시스템/기계적 트레이딩 시스템에
- 가격대와 변동성이 다른 종목 포지션을 비교할 때

### 고정 손절 대비 장점

1. 저변동성 종목은 더 큰 포지션을 가짐(가격 대비 손절 폭이 더 좁음)
2. 고변동성 종목은 더 작은 포지션을 가짐(넓은 손절이 노이즈를 흡수)
3. 종목 가격과 무관하게 포트폴리오 전반 리스크를 정규화

## Kelly Criterion

### 개념

Kelly Criterion은 승률과 손익비가 알려져 있을 때 리스크를 걸어야 할 자본의 수학적 최적 비율을 계산합니다. Bell Labs의 John L. Kelly Jr.(1956)가 개발했습니다.

### 공식

```
R         = avg_win / avg_loss       (payoff ratio)
kelly_pct = W - (1 - W) / R          (full Kelly percentage)
half_kelly = kelly_pct / 2            (practical recommendation)
```

여기서 W = 과거 승률(0~1)입니다.

### Full Kelly vs. Half Kelly

**Full Kelly**는 장기 기하성장률을 최대화하지만 변동성이 극단적으로 커집니다. 50%+ drawdown이 흔합니다. 어떤 프로 펀드도 Full Kelly를 쓰지 않습니다.

**Half Kelly**는 이론 성장률의 약 75%를 달성하면서 drawdown을 크게 낮춥니다. 실전 트레이딩의 표준 권장안입니다.

| Metric | Full Kelly | Half Kelly | Quarter Kelly |
|--------|-----------|-----------|---------------|
| 성장률 | 100% | ~75% | ~50% |
| 최대 drawdown | 심각 (50%+) | 중간 (25-35%) | 완만 (15-20%) |
| 실무 사용 | 사용 안 함 | 공격적 | 보수적 |

### 예시

- 승률: 55%, 평균 이익: $2.50, 평균 손실: $1.00
- R = 2.5 / 1.0 = 2.5
- Kelly = 0.55 - 0.45 / 2.5 = 0.55 - 0.18 = 0.37 = **37%**
- Half Kelly = **18.5%**
- $100,000 계좌 기준: 리스크 예산 = $18,500

### 음의 기대값

Kelly 공식 결과가 음수면 시스템의 기대값이 음수입니다. Kelly 비율은 0%로 바닥 처리하며, 의미는 "이 시스템은 거래하지 않는다"입니다.

예시:
- 승률: 30%, 평균 이익: $1.00, 평균 손실: $1.50
- R = 1.0 / 1.5 = 0.667
- Kelly = 0.30 - 0.70 / 0.667 = 0.30 - 1.05 = **-0.75** -> **0%**로 바닥 처리

### 두 가지 사용 모드

1. **Budget Mode** (진입가 없음): 계좌 대비 권장 리스크 예산 비율을 반환합니다. 구체적인 진입 후보를 정하기 전 자본 배분 계획에 유용합니다.

2. **Shares Mode** (진입가/손절가 포함): half-Kelly 예산을 진입-손절 거리 기반의 구체적인 주식 수로 변환합니다.

### 사용 시점

- 신뢰 가능한 과거 승률/손익 통계(최소 100+ 트레이드)가 있을 때
- 여러 전략 간 포트폴리오 수준 자본 배분에
- 상한 점검용: "내가 Kelly 제안보다 더 많이 리스크를 지고 있나?"
- 트랙 레코드가 없는 재량 트레이더에는 부적합

## 포트폴리오 제약

### 최대 포지션 크기

단일 포지션을 계좌 가치의 일정 비율로 제한합니다:

```
max_shares = int(account_size * max_position_pct / 100 / entry_price)
```

**가이드라인:**
- 5-10%: 보수적 (분산 포트폴리오, 10-20개 포지션)
- 10-15%: 중간 (집중 포트폴리오, 7-10개 포지션)
- 15-25%: 공격적 (고확신 포트폴리오, 4-7개 포지션)
- > 25%: 투기적 (대부분 트레이더에게 비권장)

### 섹터 집중도

단일 섹터 총 노출을 제한합니다:

```
remaining_pct = max_sector_pct - current_sector_exposure
remaining_dollars = remaining_pct / 100 * account_size
max_shares = int(remaining_dollars / entry_price)
```

**가이드라인:**
- 개별 종목: 포트폴리오의 5-10%
- 단일 섹터: 최대 25-30%
- 상관된 포지션: 하나의 노출로 간주

### 포지션 수와 분산

| Positions | 분산 수준 | 비고 |
|-----------|----------------|-------|
| 1-4 | 매우 집중 | 높은 변동성, 높은 확신 필요 |
| 5-10 | 집중형 | 액티브 트레이더의 최적 구간 |
| 10-20 | 분산형 | 15개 이상부터 한계효용 감소 |
| 20+ | 과도한 분산 | edge 희석, 인덱스 성과에 수렴 |

연구에 따르면 비상관 포지션 20개를 넘으면 추가 분산 이점은 미미합니다. 액티브 트레이더에게는, 확신이 희석된 20+ 포지션보다 적절히 사이징된 5-10 포지션이 위험조정수익 측면에서 더 나은 경우가 많습니다.

### Binding Constraint 로직

여러 제약이 동시에 적용될 때는 가장 엄격한 조건(최소 주식 수)이 우선합니다. 포지션 사이저는 어떤 제약이 "binding"인지 식별해 트레이더가 무엇이 포지션을 제한하는지 이해할 수 있게 합니다.

우선순위:
1. 리스크 기반 주식 수(Fixed Fractional, ATR, Kelly에서 계산)
2. 최대 포지션 % 제한
3. 최대 섹터 % 제한
4. 최종값 = 모든 후보 중 최소값

## 방법 비교

| Feature | Fixed Fractional | ATR-Based | Kelly Criterion |
|---------|-----------------|-----------|-----------------|
| 필요한 입력 | 진입가, 손절가, 리스크 % | 진입가, ATR, 배수, 리스크 % | 승률, 평균 이익/손실 |
| 변동성 반영 | 아니오 | 예 | 아니오 (과거 통계 사용) |
| 트랙 레코드 필요 | 아니오 | 아니오 | 예 (100+ trades) |
| 적합 대상 | 재량 트레이드 | 시스템/기계적 매매 | 자본 배분 |
| 복잡도 | 낮음 | 중간 | 중간 |
| 일반적 용도 | 주력 사이징 | 주력 사이징 | 상한 점검 / 배분 |
| 손절 결정 방식 | 차트 분석 | ATR 계산 | 외부 결정(차트 또는 ATR) |

### 권장 워크플로우

1. 새 전략/새로운 시장 국면에서는 **Fixed Fractional 1% 리스크**로 시작
2. 변동성 프로필이 다른 기회를 비교할 때 **ATR-based**로 전환
3. 100+ 트레이드 기록 누적 후 **Kelly를 상한선**으로 사용
4. 마지막 필터로 **제약 조건(포지션 한도, 섹터 한도)**을 항상 적용
5. 연속 손실 후 **리스크 축소**(Minervini의 "progressive exposure"를 역으로 적용)

## 리스크 관리 원칙

### 1% Rule

단일 트레이드에서 계좌 자본의 1%를 초과해 리스크를 지지 않습니다. 이는 불가피한 연속 손실 구간에서도 생존을 보장합니다:

- 1% 리스크로 10연패 = 9.6% drawdown (회복 가능)
- 5% 리스크로 10연패 = 40.1% drawdown (치명적)
- 10% 리스크로 10연패 = 65.1% drawdown (계좌 위협)

### Portfolio Heat

모든 포지션의 총 오픈 리스크는 계좌의 6-8%를 넘지 않아야 합니다:

```
portfolio_heat = sum(shares_i * risk_per_share_i) / account_size * 100
```

portfolio heat가 8%를 넘으면 기존 트레이드가 breakeven으로 이동하거나 청산될 때까지 새 포지션을 추가하지 않습니다.

### 손실의 비대칭성

손실을 회복하려면 비례 이상으로 큰 수익이 필요합니다:

| Loss | 회복에 필요한 수익 |
|------|----------------|
| 10% | 11.1% |
| 20% | 25.0% |
| 30% | 42.9% |
| 50% | 100.0% |
| 75% | 300.0% |

이 비대칭성 때문에 포지션 사이징과 손절이 종목 선택보다 더 중요합니다.
