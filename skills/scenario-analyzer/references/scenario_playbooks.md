# Scenario Playbooks

이 레퍼런스는 18개월 시나리오를 구축하기 위한 템플릿과 모범 사례를 제공합니다.
시나리오 분석 시 일관성 있고 높은 품질의 시나리오를 작성하기 위해 사용합니다.

## 시나리오 구축의 기본 원칙

### 1. MECE 원칙 (Mutually Exclusive, Collectively Exhaustive)

시나리오는 다음 조건을 충족해야 합니다:
- **상호 배타**: 각 시나리오는 중복되지 않음
- **포괄적**: 주요 가능성을 모두 커버

### 2. 확률 배분 가이드라인

| 시나리오 | 전형적 범위 | 배분 근거 |
|---------|----------|-----------|
| Base Case | 50-65% | 가장 개연성이 높은 전개 |
| Bull Case | 15-25% | 긍정적 상승 여력 |
| Bear Case | 20-30% | 부정적 하락 여력 |
| 합계 | 100% | 반드시 100%가 되도록 조정 |

**비대칭 배분이 적절한 경우:**
- Bull > Bear: 강세 재료가 많은 환경
- Bear > Bull: 리스크 요인이 많은 환경
- Base > 60%: 불확실성이 낮은 상황
- Base < 50%: 불확실성이 극도로 높은 상황 (Base Case도 불확실)

### 3. 타임라인 구분

**3단계 구조:**
- **0-6개월**: 단기적 반응, 초기 움직임
- **6-12개월**: 중기적 전개, 트렌드 형성
- **12-18개월**: 장기적 귀결, 새로운 균형

## 이벤트 유형별 시나리오 플레이북

### 1. 통화정책 이벤트 (금리 인상)

**Base Case (55%):**
- 예상대로 금리 인상 실시
- 시장은 대체로 선반영 완료
- 소폭 주가 하락, 채권 수익률 소폭 상승

**Bull Case (20%):**
- 금리 인상 폭이 예상보다 작음
- 비둘기파적 포워드 가이던스
- 주식시장 랠리

**Bear Case (25%):**
- 금리 인상 폭이 예상보다 큼
- 매파적 포워드 가이던스
- 주식시장 급락, 신용 스프레드 확대

### 2. 지정학 이벤트 (분쟁 발발)

**Base Case (50%):**
- 분쟁의 제한적 확대
- 단기적 원자재 가격 상승
- 수개월 내 상황 안정화

**Bull Case (15%):**
- 조기 정전/평화 합의
- 원자재 가격 정상화
- 시장은 조기 회복

**Bear Case (35%):**
- 분쟁의 장기화/확대
- 원자재 공급의 중대한 혼란
- 글로벌 인플레이션 가속, 경기 침체 리스크

### 3. 기술 변화 (AI 규제)

**Base Case (50%):**
- 완만한 규제 도입
- 업계 자율규제가 주체
- 혁신에 대한 영향 제한적

**Bull Case (25%):**
- 규제가 업계에 유리한 형태로 수립
- 규제 명확화로 오히려 투자 가속
- 대기업에 유리한 진입장벽 형성

**Bear Case (25%):**
- 엄격한 규제 도입
- AI 개발의 대폭 제한
- 미국 기업의 경쟁력 저하

### 4. 기업 이벤트 (대형 M&A)

**Base Case (60%):**
- 규제 당국의 승인 획득
- 예정대로 클로징
- 통합 시너지의 단계적 실현

**Bull Case (15%):**
- 예상을 상회하는 시너지 실현
- 통합이 순조로움
- 추가 M&A 전략 성공

**Bear Case (25%):**
- 규제 당국의 저지 또는 조건부 승인
- 통합의 지연/실패
- 시너지 미달성

## 시나리오 품질 체크리스트

### 내부 정합성
- [ ] 각 시나리오의 가정 조건은 논리적으로 정합한가
- [ ] 타임라인 전개의 인과관계가 명확한가
- [ ] 경제 지표 예상은 상호 정합한가

### 외부 타당성
- [ ] 과거 유사 사례와 정합적인가
- [ ] 현재 시장 환경을 적절히 반영하고 있는가
- [ ] 전문가 견해와 크게 괴리되지 않는가

### 실용성
- [ ] 투자 판단에 도움이 되는 구체성이 있는가
- [ ] 모니터링 가능한 촉매가 특정되어 있는가
- [ ] 무효화 시그널이 명확한가

### 포괄성
- [ ] 주요 리스크 시나리오가 포함되어 있는가
- [ ] 상승 가능성이 적절히 고려되어 있는가
- [ ] 테일 리스크에 대한 언급이 있는가

## 흔한 실수와 회피 방법

### 1. 현상유지 편향
**문제**: Base Case에 과도한 확률 배정 (70% 이상)
**회피**: 역사적으로 "아무것도 변하지 않을" 확률은 낮다는 것을 인식

### 2. 최신 편향
**문제**: 최근 이벤트의 영향을 과대평가
**회피**: 장기적 관점 유지, 과거 패턴 참조

### 3. 확증 편향
**문제**: 헤드라인에 부합하는 해석만 채용
**회피**: 의도적으로 반대 의견을 탐색

### 4. 과도한 정밀도
**문제**: 18개월 후 수치를 소수점 이하까지 예상
**회피**: 불확실성을 인정하고 범위로 표현

### 5. 시나리오 중복
**문제**: Base/Bull/Bear가 부분적으로 겹침
**회피**: 각 시나리오의 경계 조건을 명확화

## 확률 업데이트 가이드라인

새로운 정보가 들어왔을 때의 확률 조정:

| 새 정보의 성격 | 확률 조정 방향 |
|-------------|---------------|
| 시나리오를 지지하는 데이터 | +5-15% |
| 시나리오에 반하는 데이터 | -5-15% |
| 결정적 증거 | +20-30% or -20-30% |
| 새로운 리스크 요인의 출현 | Bear Case +5-10% |
| 리스크 요인의 해소 | Bear Case -5-10% |

**조정 후 반드시 합계 100%로 재조정**

## 출력 품질 기준

고품질 시나리오의 특징:
1. **구체성**: 추상적이지 않고, 수치/날짜/명칭이 포함됨
2. **논리성**: 인과관계가 명확
3. **검증 가능성**: 나중에 정오를 판단할 수 있음
4. **실용성**: 투자 판단에 직결되는 정보를 포함
5. **겸허함**: 불확실성을 적절히 표현
