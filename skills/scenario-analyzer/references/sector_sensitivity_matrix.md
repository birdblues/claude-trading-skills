# Sector Sensitivity Matrix

이 레퍼런스는 각종 이벤트 유형에 대한 섹터별 민감도를 매트릭스 형식으로 정리하고 있습니다.
시나리오 분석 시 어떤 섹터가 영향을 받기 쉬운지 빠르게 판단하기 위해 사용합니다.

## 범례

**영향도:**
- `++` : 강한 긍정적 영향
- `+` : 긍정적 영향
- `0` : 중립/영향 경미
- `-` : 부정적 영향
- `--` : 강한 부정적 영향

**확신도:**
- `H` : 높음 (과거 패턴이 일관)
- `M` : 중간 (상황 의존)
- `L` : 낮음 (불확실성 높음)

## 사용 방법

1. **이벤트 유형 특정**: 헤드라인에서 이벤트 카테고리를 판단
2. **해당 매트릭스 참조**: 위에서 적절한 매트릭스를 선택
3. **영향도와 확신도 확인**: 섹터별 영향을 파악
4. **대표 종목을 출발점으로**: 심층 분석의 기점으로 사용
5. **복합 시나리오는 복수 참조**: 여러 이벤트가 얽힌 경우 복수 매트릭스를 통합

## 주의사항

- **개별 종목 상황**: 섹터 영향과 개별 종목 영향은 다를 수 있음
- **타이밍**: 즉시 영향과 지연 영향을 구분
- **규모감**: 이벤트의 규모에 따라 영향도는 변동
- **시장 선반영도**: 이미 시장이 선반영한 경우 반응이 제한적
