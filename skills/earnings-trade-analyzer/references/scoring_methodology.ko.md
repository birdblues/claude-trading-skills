# Earnings Trade Analyzer - 점수 산정 방법론

## 개요

Earnings Trade Analyzer는 5요인 가중 점수 시스템으로 실적 발표 후 주식 셋업을 평가합니다. 각 종목은 종합 점수(0-100)와 문자 등급(A/B/C/D)을 받습니다.

## 5요인 점수 시스템

### 요인 1: Gap Size (가중치 25%)

방향과 무관하게 실적 갭의 크기를 측정합니다. 갭이 클수록 기관의 확신이 강하다는 신호입니다.

| Absolute Gap | Score |
|-------------|-------|
| >= 10%      | 100   |
| >= 7%       | 85    |
| >= 5%       | 70    |
| >= 3%       | 55    |
| >= 1%       | 35    |
| < 1%        | 15    |

**타이밍 로직:**
- **BMO (Before Market Open):** gap = open[earnings_date] / close[previous_day] - 1
- **AMC (After Market Close):** gap = open[next_day] / close[earnings_date] - 1
- **Unknown:** 기본값으로 AMC 로직 사용

### 요인 2: Pre-Earnings Trend (가중치 30%)

실적 발표 전 20일 가격 수익률을 측정합니다. 긍정적 실적 갭 이전에 상승 추세였던 종목은 더 강한 모멘텀 특성을 보입니다.

| 20-Day Return | Score |
|--------------|-------|
| >= 15%       | 100   |
| >= 10%       | 85    |
| >= 5%        | 70    |
| >= 0%        | 50    |
| >= -5%       | 30    |
| < -5%        | 15    |

### 요인 3: Volume Trend (가중치 20%)

실적 날짜 주변의 20일 평균 거래량 대비 60일 평균 거래량 비율입니다. 비율이 높을수록 기관 관심 및 축적 신호가 강합니다.

| Volume Ratio (20d/60d) | Score |
|------------------------|-------|
| >= 2.0x                | 100   |
| >= 1.5x                | 80    |
| >= 1.2x                | 60    |
| >= 1.0x                | 40    |
| < 1.0x                 | 20    |

### 요인 4: MA200 Position (가중치 15%)

200일 Simple Moving Average 대비 현재 가격 위치를 평가합니다. 200일 SMA 위 종목은 장기 상승 추세에 있습니다.

| Distance from MA200 | Score |
|---------------------|-------|
| >= 20%              | 100   |
| >= 10%              | 85    |
| >= 5%               | 70    |
| >= 0%               | 55    |
| >= -5%              | 35    |
| < -5%               | 15    |

### 요인 5: MA50 Position (가중치 10%)

50일 Simple Moving Average 대비 현재 가격 위치를 평가합니다. 50일 SMA 위 종목은 중기 강세를 보입니다.

| Distance from MA50 | Score |
|--------------------|-------|
| >= 10%             | 100   |
| >= 5%              | 80    |
| >= 0%              | 60    |
| >= -5%             | 35    |
| < -5%              | 15    |

## 등급 임계값

| Grade | Score Range | Description |
|-------|-----------|-------------|
| A     | 85-100    | 기관 축적이 동반된 강한 실적 반응 |
| B     | 70-84     | 모니터링할 가치가 있는 양호한 실적 반응 |
| C     | 55-69     | 신호가 혼재되어 주의 필요 |
| D     | 0-54      | 약한 셋업, 회피 권장 |

## Entry Quality Filter

`--apply-entry-filter`가 활성화되면 다음 종목이 제외됩니다:
- 현재 가격이 $10 미만인 종목(페니 주 영역, 기관 관심 낮음)

## 과거 승률 컨텍스트

이 점수 시스템은 실적 발표 후 모멘텀이 지속될 확률이 가장 높은 종목을 식별하도록 설계되었습니다. 핵심 관찰:

- **Grade A stocks** (85+)는 과거 기준으로 가장 강한 post-earnings drift (PEAD)를 보임
- **Pre-earnings trend**는 실적 전 모멘텀이 실적 후 추세 지속의 가장 강한 예측 변수이므로 가장 높은 가중치(30%)를 가짐
- **Volume confirmation** (20%)은 실적 움직임에 기관 참여가 동반됐는지 검증
- **Moving average positions** (합산 25%)은 종목이 우호적인 추세 구조에 있는지 확인

## 종합 점수 공식

```
composite_score = (gap_score * 0.25) + (trend_score * 0.30) + (volume_score * 0.20)
                + (ma200_score * 0.15) + (ma50_score * 0.10)
```

가중치 합은 정확히 1.0 (100%)입니다.
