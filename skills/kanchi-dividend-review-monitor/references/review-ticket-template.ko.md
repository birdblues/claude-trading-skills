# Review Ticket Template

각 `REVIEW` 티커에 이 템플릿을 사용합니다.

```markdown
# REVIEW Ticket: [Ticker]

## Trigger Summary
- Final status: REVIEW
- Trigger hits: [T1,T4]
- Generated at:

## Evidence (One Line Each)
- T1:
- T2:
- T3:
- T4:
- T5:

## Manual Checks Required
1. 최신 공시/보도자료의 1차 출처 원문을 확인합니다.
2. 이슈의 범위와 지속 기간을 확인합니다.
3. 배당 지속성에 대한 영향을 평가합니다.
4. payout/coverage 가정을 재계산합니다.
5. 포트폴리오 조치(hold / trim / exit review)를 결정합니다.

## Decision Log
- Decision:
- Rationale:
- Owner:
- Due date:
```
