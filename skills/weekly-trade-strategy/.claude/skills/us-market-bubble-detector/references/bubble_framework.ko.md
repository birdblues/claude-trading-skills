## 핵심 개념

### 버블의 본질: 사회적 전염을 통한 사회 규범의 전도

진짜 버블은 **가격 수준**이 아니라 **군중 심리의 단계**로 판단해야 합니다.

#### 핵심 특성

1. **임계 정보 폭포(Information Cascade)**
   - 전 계층(비투자자 포함)으로 확산이 완료됨
   - FOMO(Fear of Missing Out)가 사회 규범이 됨
   - 회의적인 태도를 취할 사회적 비용이 최대화됨

2. **사회적 계산의 전도**
   ```
   Normal state: Conformity pressure < Value of independent judgment
   Bubble state: Conformity pressure > Value of independent judgment
   ```
   "동조하지 않을 고통 < 역행 관점을 유지할 고통"이 되는 시점이 완성에 가깝습니다.

3. **제도화된 확증 편향**
   - 미디어, 전문가, 일반 대중이 동일 내러티브를 반복
   - 반증은 무시되거나 배제됨
   - "This time is different"가 주문처럼 반복됨

### 신호 예시: 택시기사의 투자 이야기

1929년 대폭락 직전 구두닦이 소년의 주식 이야기를 듣고 매도했다는 Joe Kennedy 일화의 현대판입니다.

**왜 이것이 결정적 신호인가:**
- 정보가 최하위 계층까지 도달 = cascade 완료
- "마지막 매수자" 집단 진입 = 수요 고갈 임박
- 전문성 없는 "확실한 수익" 인식 = 리스크 무지의 정점

## Minsky/Kindleberger 모델

5단계 버블 진행 모델 (Hyman Minsky / Charles Kindleberger)

### 1. Displacement (Trigger)

**특징:**
- 신기술, 제도 개혁, 통화 완화 같은 구조적 변화
- 정당한 투자 기회가 등장
- 합리적 가격 상승

**실제 예시:**
- 1990년대: 인터넷 혁명
- 2010년대: 모바일/클라우드
- 2020-21년: 팬데믹 대응 초완화 + 원격 기술

### 2. Boom (Expansion)

**특징:**
- 자기강화 루프: 가격 상승 → 미디어 노출 → 신규 참여자 → 유동성 확장
- 전문가의 긍정적 시각 증가
- 밸류에이션은 높지만 "성장 기대"로 설명 가능

**심리:**
- 가용성 편향: 성공 사례가 지배
- 군집 행동: 기관투자자도 합류 압박을 느낌

**탐지 지표:**
- 거래대금의 지속적 증가
- 신규 계좌 개설 가속
- 연간 수익률 75-90퍼센타일

### 3. Euphoria (Exuberance)

**특징:**
- 내러티브가 "상식"이 되고 반대 의견은 "구시대적"으로 낙인
- 레버리지 증가 (신용, 선물, 파생)
- 신규 발행 급증 (IPO/ICO/SPAC)
- 저품질 종목까지 동반 상승

**심리:**
- 과신: "나는 특별하다"
- 확증 편향: 유리한 정보만 수용
- 후회 회피: "매도 후 상승을 놓칠 후회" 과대평가

**탐지 지표:**
- VIX 하락 (리스크 경시)
- Put/Call 극단 편향
- Margin balance 사상 최고
- "XX-related" 상품 범람

### 4. Profit Taking (Exit Begins)

**특징:**
- 초기 참여자(smart money)가 수익 실현 시작
- 군중은 FOMO로 추격 지속
- 변동성(가격 흔들림) 확대

**심리적 분기:**
- Smart money: 손실 회피 (이익 확보 우선)
- 군중: FOMO 정점 ("놓치면 안 돼")

**탐지 지표:**
- 거래량 급증 + 가격 변동 폭 확대
- 내부자 매도 증가
- 공매도 잔고 증가 (정교한 회의론)

### 5. Panic (Reversal)

**특징:**
- 추세 붕괴의 객관적 확인 (신고가 실패, 이평선 이탈)
- 강제 청산 → 청산 연쇄
- 유동성 증발

**심리:**
- 손실 회피의 역전: "손실 확정 회피" → "추가 손실 회피를 위한 공황 매도"
- 군집 행동 역전: 매수 군중 → 매도 군중

**탐지 지표:**
- 서킷 브레이커 발동
- 마진콜 연쇄
- MTM 손실 사상 최악 수준

## 행동심리 요소

### 1. FOMO (Fear of Missing Out)

**메커니즘:**
- 사회적 증거: "모두가 산다 → 맞는 것 같다"
- 후회 회피: "기회를 놓친 후회" > "돈 잃은 후회"

**버블 조건:**
FOMO가 개인 심리에서 **사회 규범**으로 승격 → 비동조가 직업/사회적 리스크가 됨

### 2. 확증 편향

**버블 시기 증폭 요인:**
- 미디어 에코챔버
- 소셜미디어 알고리즘 필터 버블
- 전문가 동조 압력 (역행 = 커리어 리스크)

### 3. 과신

**버블 시기 특징:**
- "나는 꼭대기에서만 팔 수 있다"
- "이번에는 다르다"
- 리스크 관리 경시

### 4. 위험한 조합: Loss Aversion × Regret Aversion

**가장 위험한 단계:**
"초기 수익 실현 후 급등"을 경험한 투자자

1. 수익 실현 → 가격 추가 상승 → 후회
2. 재진입 → 고점 매수 → 손실 회피로 손절 불가
3. 추가 상승 → "내가 맞았다" 착각 강화
4. 반전 시점에서 탈출 실패

## 탐지를 위한 정량 지표

### Category 1: 사회적 침투

| Indicator | Data Source | Alert Threshold |
|-----------|------------|----------------|
| Google Search Trends | Google Trends API | 5x+ normal |
| Social Media Mentions | Twitter/Reddit API | z-score > +3 |
| New Account Openings | Brokerage data | 200%+ YoY |
| Media Coverage | News APIs | Weekly article count 10x normal |

### Category 2: 가격 다이내믹스

| Indicator | Calculation | Alert Threshold |
|-----------|------------|----------------|
| Annualized Return | Annualize 90-day return | Exceeds 95th percentile |
| Price Acceleration | 2nd derivative sign and magnitude | Positive and increasing |
| Volatility Skew | Put/Call ratio | < 0.5 (extreme optimism) |
| Distance from 52W High | (Current - 52W High) / 52W High | Within -5%, clustering |

### Category 3: 레버리지 및 포지셔닝

| Indicator | Data Source | Alert Threshold |
|-----------|------------|----------------|
| Margin Balance | FINRA | All-time high |
| Mark-to-Market P&L | Exchange data | Extreme unrealized gains (reversal risk) |
| Futures Positioning | CFTC COT | Speculators extremely long |
| Funding Rate | Crypto exchanges | 50%+ annual sustained |

### Category 4: 신규 발행 및 신규 유입

| Indicator | Data Source | Alert Threshold |
|-----------|------------|----------------|
| IPO Count | Renaissance IPO Index | 100%+ YoY |
| SPAC Formation | SPAC statistics | 100+ per quarter |
| Theme ETF Launches | ETF.com | 5+ "XX-related" per month |

### Category 5: 밸류에이션

| Indicator | Calculation | Alert Threshold |
|-----------|------------|----------------|
| Shiller CAPE | P/E using 10-year real earnings average | >30 (historically high) |
| Buffett Indicator | Market cap / GDP | >150% (overheating) |
| Sector Divergence | Top sector vs bottom sector P/E difference | 3x+ gap |

## 실전 대응 전략

### 공격: 수익 실현 전략

#### 1. 기계적 계단식 수익 실현

```
Target Return    Profit %    Remaining Position
+20%             25%         75%
+40%             25%         50%
+60%             25%         25%
+80%             25%         0%
```

**장점:**
- 심리적 압박 제거
- "매도 후 상승" 후회 완화
- 부분 수익 확보 보장

#### 2. 시간 분산 청산

**NG 패턴:**
특정 이벤트(실적, 제품 출시, 지수 편입)에 수익 실현 집중

**권장:**
- 2주 동안 매일 10%씩 매도
- 이벤트 전/중/후로 분산

#### 3. Trailing Stop (변동성 조정)

```python
stop_price = current_price - (ATR × coefficient)
# ATR: Average True Range (20-day average fluctuation)
# Coefficient: 2.0 (normal), 1.5 (bubble zone, tightened)
```

### 방어: 리스크 관리

#### 1. 사전 정의된 Drawdown 허용치

```
Expected Max DD    Position Size
-10%               100% (full position)
-20%               50%
-30%               33%
-40%               25%
```

#### 2. 버블 단계별 리스크 예산 (REVISED v2.1)

| Bubble Stage | Score | Total Risk Budget | New Entry | Stop Coefficient |
|-------------|-------|------------------|-----------|-----------------|
| Normal | 0-4 | 100% | Normal | 2.0 ATR |
| Caution | 5-7 | 70-80% | 50% reduced | 1.8 ATR |
| Elevated Risk | 8-9 | 50-70% | Selective | 1.6 ATR |
| Euphoria | 10-12 | 40-50% | Stop | 1.5 ATR |
| Critical | 13-15 | 20-30% | Stop | 1.2 ATR |

**v2.1 핵심 변경:**
- 더 세분화된 리스크 관리를 위해 "Elevated Risk" 단계(8-9점) 추가
- 9점 구간 리스크 예산을 과도하지 않게 조정
- 최대 점수 15로 축소 (Phase 2: 최대 12, Phase 3: 엄격 기준으로 최대 3)

#### 3. 공매도 타이밍 (중요)

**NG 패턴:**
- 주관적으로 "너무 많이 올랐다"는 이유의 조기 숏
- 버블은 "예상보다 오래 지속"될 수 있음 ("Markets can remain irrational longer than you can remain solvent")

**권장 조건 (복합):**
1. 주봉 차트에서 lower highs가 명확
2. 거래량이 피크아웃(하락 추세 진입)
3. funding rate 급락(crypto) / margin balance 하락(stocks)
4. 미디어/검색 트렌드 피크아웃
5. 섹터 내 약한 종목부터 선행 붕괴

최소 3개 충족 시 시작 검토.

#### 4. 현금 계좌 분리

- **장기 투자 계좌**: Buy & hold, 배당 재투자, 리밸런싱만
- **트레이딩 계좌**: 단기 트레이딩, 버블 수익 실현, 반전 숏

**목적:** 의사결정 혼선을 방지하고 심리적 안정 유지

### 실전 일일 체크리스트

장 시작 전 매일 아침 점검:

- [ ] Bubble-O-Meter 업데이트 (8개 지표 점수화)
- [ ] 보유 포지션 ATR trailing stop 업데이트
- [ ] 현재 버블 단계에서 신규 진입 계획이 적절한지 확인
- [ ] 미디어/소셜 트렌드 급변 여부 점검
- [ ] 주요 지수의 52주 고점 대비 거리 확인
- [ ] 레버리지 지표(margin balance, funding rate) 점검
- [ ] VIX 수준과 Put/Call ratio 점검
- [ ] 섹터 breadth 점검 (광범위 상승인지 선택적 상승인지)

## 요약: 실전 황금 규칙

1. **"가격이 아니라 과정을 보라"**
   버블은 레벨이 아니라 군중 심리 단계 전환으로 평가한다

2. **"택시기사가 주식을 말하면 나와라"**
   마지막 매수자 집단 진입 = 수요 고갈 임박

3. **"'이번에는 다르다'는 늘 같다"**
   "이번에는 다르다"가 주문이 되면 매우 조심해야 한다

4. **"기계적 규칙이 심리를 보호한다"**
   동조 압력이 정점일수록 사전 규칙 준수가 생명선이다

5. **"확인 후 숏, 수익 실현은 일찍"**
   버블 붕괴는 늦게 오지만 급격하다. 역행 숏은 위험하다. 수익은 기계적으로.

6. **"회의가 아파질 때 끝이 시작된다"**
   사회적 비용이 독립 판단을 넘는 순간이 임계점이다
